## 配置目录

- config.json
    - npm 包管理工具，可选yarn|npm|cnpm
    - db
      - host 主机
      - port 端口号
      - username 用户名
      - password 密码
- key.json
    - id": 作者id,
    - privKey: 作者秘钥,
    - pubKey: 作者公钥,
    - username: 用户名,
    - nickname: 昵称,
    - realname: 真是姓名,
    - mobile: 手机号码,
    - email: 邮箱,
    - website: 官网,
    - avatar: 头像
